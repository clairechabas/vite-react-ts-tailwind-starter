## Getting Started
1. Clone this repo
2. Install dependencies by running `yarn`

That's it!

<br />

## Available Scripts

Start the development server:
```bash
yarn dev
```

Build your app for production:
```bash
yarn build
```

Preview the production version you just built (result of latest `yarn build`):
```bash
yarn preview
```
